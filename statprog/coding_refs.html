Coding Practices: Suggested Reading

  • Jon Bentley (2000) Programming Pearls, 2nd edition. Addison-Wesley.

  • Steve Oualline (1992) C Elements of Style. M&T Publishing.

  • Kernighan and Pike (1999) The Practice of Programming. Addison-Wesley.

  • Kernighan and Plauger (1978) The Elements of Programming Style. McGraw-Hill.

  • Steve McConnell (1993) Code Complete. Microsoft Press.

  • Steve McGuire (1993) Writing Solid Code. Microsoft Press.
