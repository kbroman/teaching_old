Statistics for laboratory scientists (2)

Homework problems for lecture 2


This homework tries to clarify when asymptotic tests such as the likelihood-ratio test or the chi-square test are not appropriate. But remember, randomization tests can bail you out!

Asymptotic tests might not be appropriate when you have only a small number of observations. Assume you have the hypothesis that the underlying ratio of cases in a genetic experiment are 9:3:3:1. Consider six possibilities: the number of data points that you can observe being 5, 10, 20, 40, 80, or 160. For each of those six possibilities, simulate 10,000 observations from a multinomial distribution with the appropriate probabilities according to the assumed ratio (for example rmultinom(n=5,p=(9/16,3/16,3/16,1/16)). For each observation in each of the six groups (n=5,...,160), calculate the chi-square test statistic.

  1. Plot a histogram of the test statistics for each of those six groups. Comment.

  2. Explain why we can not use the likelihood-ratio test statistic to generate those plot.
